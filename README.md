# Vitest-learning

Exercices et pratiques autour du testing avec Vitest

## 🎯 Objectif

Approfondir la compréhension :

* de la syntaxe Vitest
* des matchers et assertions
* des tests synchrones et asynchrones
* des mocks et spies
* de l’environnement Vite et du testing moderne
* des bonnes pratiques d’écriture de tests rapides et maintenables

## 📂 Organisation

Les dossiers suivent ma progression :

* les premiers exercices reflètent mes débuts avec Vitest
* les plus récents montrent une amélioration dans la précision, la structure et la qualité des tests
* certains exemples comparent les approches Jest vs Vitest pour mieux comprendre les différences

Ce repository est volontairement évolutif et documente mon apprentissage.

## ⚡ Pourquoi Vitest ?

* exécution rapide grâce à Vite
* configuration simplifiée
* API proche de Jest → prise en main rapide
* meilleure intégration avec les projets modernes (Vue, React, etc.)

## 🛠 Stack

* JavaScript / TypeScript
* Vitest
* Vite
